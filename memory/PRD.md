# JIS Terraplanagem e Pavimentação Asfáltica - PRD

## Problema Original
Site institucional moderno, profissional e responsivo para a empresa JIS Terraplanagem e Pavimentação Asfáltica LTDA.

## Arquitetura
- **Frontend**: React + TailwindCSS (já configurado no template)
- **Componentes**: Shadcn/UI
- **Design**: Industrial/Corporativo com paleta Azul Escuro (#0f172a), Laranja (#f97316), Cinza (#374151)
- **Tipografia**: Barlow Condensed (headings) + Inter (body)

## User Personas
1. **Empresas de Construção**: Buscam parceiros para obras de infraestrutura
2. **Fazendeiros/Agronegócio**: Precisam de terraplanagem e barragens para irrigação
3. **Prefeituras**: Necessitam de pavimentação e obras públicas
4. **Construtoras/Loteadoras**: Preparação de terrenos para loteamentos
